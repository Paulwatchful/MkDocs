## Register Procedure

1. Select Register
2. Add Email address to Email input box
3. Add Password to input box and confirm by adding again to the second box
4. Click Register 

  

### Possible Issues

- Incorrect email address
- Passwords not matching
- Password manager auto populating the boxes
- Caps lock on 
- Num Lock off
